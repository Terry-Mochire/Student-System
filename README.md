## Lore Learning Management System

#### Description 
A student management system is a system that gives you a bird's-eye view of how an entire educational institution operates. It is a management information system that assists in the management of various processes in an educat ional institution such as General Administration, Staff Management, Academics, Student Management, and Accounts, among others. The information is created using cutting-edge technologies, making decision making much faster, more effective, and simpler than ever before. Also contributes to the institution's overall educational quality.

## Setup Informartion

#### Prerequisites
* Basic Git knowledge, including an installed version of Git.
* Your application must run on the OpenJDK version 6, 7 or 8.

#### Technologies Used
1. Java
2. Spark
3. Gradle
4. jUnit
5. Handlebars
6. PostgreSQL
7. HTML, CSS,Bootstrap


To run the application, in your terminal:


    1. Clone or download the Repository
    2. cd into directory then run `cd StudentSytem`
    3. Rebuild the Project Using Intellij IDEA or ...
    4. open psql
    6. Run commands provided in src/main/resources/db/create.sql to create your database and required tables
